# OpenVPNGateMonitor.SharedModels

Shared models for the OpenVPNGateMonitor project.  
These models are used for API communication between backend services and consumers.

## Features

- DTOs for OpenVPN Gate Monitor API
- Simple and lightweight
- Compatible with .NET 6 7 8 9

## Usage

Install via NuGet:

```bash
dotnet add package OpenVPNGateMonitor.SharedModels
```

Import in your code:

```csharp
using OpenVPNGateMonitor.SharedModels;
```
